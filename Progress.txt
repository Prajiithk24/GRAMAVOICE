AbstractGramavoice is an AI-powered civic grievance platform designed to simplify the process of reporting public issues, especially for rural and non-English-speaking populations. The system enables users to submit complaints through voice input in Tamil, eliminating the need for typing or technical knowledge. The spoken input is converted into text using speech recognition, after which a machine learning model automatically classifies the complaint into relevant categories such as water supply, electricity, road conditions, municipality services, and ration distribution.By automating complaint categorization and structuring, Gramavoice ensures faster processing and efficient routing of issues to the appropriate authorities. The platform aims to bridge the communication gap between citizens and government bodies, improve transparency, and enhance responsiveness in public service systems. With an intuitive interface and language accessibility, Gramavoice empowers citizens to raise their concerns easily and contributes to smarter governance through the use of artificial intelligence.ObjectivesTo develop a voice-based system for reporting civic grievances in Tamil.To convert spoken complaints into text using speech recognition.To automatically classify complaints into categories such as water, electricity, roads, municipality, and ration.To simplify the complaint submission process by removing the need for typing.To ensure efficient routing of complaints to the appropriate authorities.To make the system accessible to rural and non-technical users.ModulesUser Interface Module (mobile/web interface for users to submit complaints)Voice Input Module (records and processes Tamil speech input)Speech-to-Text Conversion Module (converts voice into Tamil text)Text Preprocessing Module (cleans and prepares text for analysis)Complaint Classification Module (AI model to categorize complaints)Category Mapping Module (maps classified output to departments like water, EB, roads, etc.)Backend/API Module (handles data processing and communication between components)Database Module (stores complaints, categories, and user data)Notification Module (updates users on complaint status)Admin Dashboard Module (interface for authorities to view and manage complaints)Gap AnalysisExisting grievance platforms primarily rely on text-based input, which creates a significant barrier for users who are not comfortable with typing or digital interfaces. These systems offer limited support for regional languages such as Tamil, making them less accessible to a large section of the population. Additionally, rural and non-technical users often struggle to navigate complex interfaces, resulting in reduced usage. Most platforms lack intelligent automation for complaint classification, requiring users to manually select categories, which can lead to errors. Furthermore, manual processing of complaints causes delays, reducing the overall efficiency and responsiveness of the grievance redressal system.Product DescriptionGramavoice is an AI-powered grievance reporting application that enables citizens to report civic issues using Tamil voice input, eliminating the need for typing or complex forms. The system converts spoken complaints into text through speech recognition and uses a machine learning model to automatically classify them into categories such as water, electricity, roads, municipality services, and ration distribution. By automating complaint processing and routing, Gramavoice ensures faster and more accurate handling of issues, making the platform accessible, user-friendly, and especially beneficial for rural and non-technical users while improving communication between citizens and authorities.Comparison (Apps Available)The MyGrievance application is a government platform that allows citizens to lodge complaints related to central and state departments and track their status through a dashboard. However, it mainly relies on text input and manual form filling, which can be difficult for non-technical users. The Jansunwai app is another grievance redressal system where users can register complaints and receive a reference number to track progress. While it provides a structured complaint tracking mechanism, it still depends on users typing and manually selecting complaint categories. The UMANG platform integrates multiple government services, including grievance submission, into a single application. Although it offers a wide range of services, the interface can be complex and lacks intelligent automation such as voice-based input and AI-driven classification.UniquenessGramavoice stands out from existing grievance platforms by introducing a voice-first, AI-driven approach tailored for regional users. Unlike current applications that depend on text input and manual category selection, Gramavoice allows users to simply speak their complaints in Tamil, making it highly accessible to rural and non-technical populations. The system automatically converts speech to text and uses machine learning to classify complaints into appropriate categories, eliminating user effort and reducing errors. This combination of regional language support, voice interaction, and intelligent automation makes Gramavoice faster, more inclusive, and significantly easier to use compared to traditional grievance reporting systems.
